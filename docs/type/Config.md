# Type summary
>### Config: Represents the SPAL configuration object.
>#### spalOptions: container object
>##### general: this object contains the general configurations
>###### homeName: represents which view will be the homepage
>###### preloadMode: represents the preload functionality
>###### isSuppressWarning: represents the suppress warning functionality
>###### isDebugMode: represents the debug mode functionality
>##### frame: this object contains the frame elements
>###### name: represents the frame element object's name
>##### view: this object contains the view elements
>###### name: represents the view element's name
>###### title: represents the view element's title
>###### preload: represents the view element's preload functionality

>### FrameElementConfig
>###### name: represents the frame element object's name

>### ViewElementConfig
>###### name: represents the view element's name
>###### title: represents the view element's title
>###### preload: represents the view element's preload functionality
