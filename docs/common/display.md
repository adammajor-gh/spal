# Function summary
>### display(layoutId: string, element: LayoutElement)
>Display the given layout element in the given layout, based on the layout's id.

>### frame(layoutNodeList: NodeListOf\<Element>, frameElements: FrameElement[])
>Display all the frame element, which is in the given list in the layouts, which is in the given nodelist.

>### view(viewElement: ViewElement)
>Display the given view element in the view layout.
