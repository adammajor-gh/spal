# Function summary
>### layoutScan(): NodeListOf\<Element>
>Return a NodeListOf\<Element>
>This function searches all the element in the current frame/view, and return the elements, which have the 'spal-layout...' id. (basically all the layout elements)

>### layoutScan(): NodeListOf\<Element>
>Return a NodeListOf\<Element>
>This function searches all the element in the current frame/view, and return the elements, which have the 'spal-vc...' id. (basically all the view changer elements)
