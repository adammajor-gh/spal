# Function summary
>### createFrameElement(frameElementConfig: FrameElementConfig): Promise\<FrameElement>
>This function can build and return a frame element object, based on the frame element's name.

>### readAll(config: Config): Promise\<FrameElement[]>
>This function searches all the frame elements which is in the config object, build a frame element object each other and return all the builded frame elements in a list.
